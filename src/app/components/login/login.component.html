Projeto criado para estudar a api node js com Angular standalone e a conexão com moongoDB.

Ele é ideal para quem busca entender melhor como integrar um back-end robusto com um front-end dinâmico, utilizando tecnologias modernas e eficientes como Node.js e Angular.
